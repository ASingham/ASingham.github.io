Rules

Here are some rules, please make sure to obey them all so we can all have fun!

  1. Tips for entering in codes:
    1. Codes are case insensitive. (e.g. "SECRETCODE" is the same as "secretcode", "SeCrEtcoDE", etc.)
    2. Spaces are ignored. (e.g. “really long code” is the same as “reallylongcode”)
    3. Punctuation matters! So remember to use it.
    4. Press enter or click the button to submit the code.
    5. If you move to the next page you have gotten the question right!
  2. Work as a team! Make sure that everyone is up to the same challenge as you before continuing.
  3. If you accidentally close your page, click the Fast Travel button at the top and ask your teammate for the Fast Travel Code.
  4. Please don’t try exploiting the website, I could’ve spent time into making a less exploitable website, but I instead used that time to create more challenges for you guys. I trust you guys to do the right thing!
  5. Feel free to call me over for assistance if you need it! I can’t guarantee I’ll give you a hint every time, so use my help sparingly!
